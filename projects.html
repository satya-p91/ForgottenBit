Food Ordering System


This is a java based project. The main purpose of projects is to inplement the concepts servelet, jsp and how to deploy WAR file in server(tomcat).

Brief detail of the application: With this a resturant can store all the details about sold foods like food name,price,customer name,mobile number,date and time.

It have two type of user ADMIN and EMPLOYEE. admin have all permissions like can register a new employee or remove a employee, show or modify all the stored records and insert record as well.
